## 使用介绍
### 软件本体功能介绍
1. ~~双击` .exe`即可打开~~
2. 打开后桌面右下角将出现默认gif
3. 左键拖动可更改小人位置
4. 右键可弹出设置框,设置框中
   * `关闭设置` 可关闭设置框
   * `关闭软件` 可关闭程序
   * `锁定` 可锁定gif现有位置
   * 下拉框 `img0` 可选择已有的gif
   * `位置重置` 可用于重置gif现有位置
